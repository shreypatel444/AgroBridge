**Connecting Farmers & Brokers for a Sustainable Marketplace**  

AgroBridge is a **Next.js web application** designed to bridge the gap between farmers and brokers, enabling direct interaction, better price transparency, and a seamless agricultural marketplace experience. It features **high-level UI/UX, responsive design, and rich functionality** for both farmers and brokers.

- **Responsive & High-Level UI** ‚Äì Built with **TailwindCSS**, **Framer Motion**, and **Lottie animations** for a smooth and modern interface.  
- **Authentication** ‚Äì Secure login and user management using **Clerk**.  
- **Dynamic Navbar** ‚Äì Shows different navigation links depending on whether the user is signed in.  
- **About Page** ‚Äì Showcases the mission and vision of AgroBridge.  
- **Contact Page** ‚Äì Allows signed-in users to reach out to support or stakeholders.  
- **Book Appointment** ‚Äì Schedule meetings between farmers and brokers.  
- **Market Explore** ‚Äì Browse agricultural markets and discover opportunities.  
- **Brokers Directory** ‚Äì List of verified brokers for easy contact and networking.  
- **Animations & Interactions** ‚Äì Smooth transitions, hover effects, and animated hero section for engaging UX.
